## 3 Practical Search & Resurfacing Queries

Quick, practical queries for finding forgotten information—essential for busy professionals juggling multiple priorities.

### 1. **Finding Forgotten Solutions**

**Query:**

```
I remember solving a problem related to [specific issue or error] a few weeks ago, but I can't remember the exact solution. Can you help me find how I solved it and what approach I used?
```

**What This Query Does:**

- **Time:** Past few weeks (flexible timeframe)  
- **Modality:** ALL sources (code, terminal, research, conversations)  
- **Topic:** Specific problem-solving solution  
- **Why It Works:** Leverages accumulated memory to find past solutions, helps avoid re-solving problems, retrieves successful approaches

**Example Results:** Returns the specific error message, debugging steps taken, code changes made, terminal commands used, and the final solution that worked.

**Best For:** Quickly resurfacing solutions to technical problems, avoiding duplicate work, maintaining consistency in problem-solving approaches.

Understanding team dynamics and optimizing workflows—critical for senior roles managing teams and processes.


**What This Query Does:**

- **Modality:** SPECIFIC - Communications and collaborative work  
- **Topic:** Team relationships and collaboration patterns

## Query Best Practices

### Effective Query Strategies:

- **Be specific:** "authentication bug fix" is better than "that bug"  
- **Mention timeframes:** "a few weeks ago" or "last quarter" helps narrow results  
- **Include context:** Mention related projects, people, or tools to help Pieces find it  
- **Use natural language:** Ask like you're talking to a colleague  
- **Combine elements:** Time + topic + modality = better results

### When to Use Each Type:

- **Search Queries (1-3):** Use daily when you need to find something specific  
- **Collaboration & Efficiency (4-5):** Use monthly to understand patterns and optimize  
- **Project Management (6-7):** Use when planning or reviewing project status  
- **Strategic Planning (8-10):** Use quarterly for performance reviews and planning
