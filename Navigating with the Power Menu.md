# Navigating with the Power Menu

## What is the Power Menu?

The Power Menu in Pieces is a dynamic command palette that gives you quick access to navigation and actions throughout the app. It's designed to keep you in the flow—whether you're working in Workstream Activity, Pieces Copilot, or any other view.

Think of it as your command center for moving around Pieces quickly and efficiently.

![](https://storage.googleapis.com/pieces_static_resources/pro_tips/power_menu_overview.png)

## How to Access the Power Menu

You can open the Power Menu in two ways:
1. **Keyboard shortcut** — Press the Power Menu hotkey (check your keyboard shortcuts in Settings)
2. **App interface** — Open it from the app interface

Once open, you can search for any command or action you need.

---

## Navigating Between Workstream Activity and Pieces Copilot

The Power Menu makes it easy to jump between Workstream Activity and Pieces Copilot—two of the most powerful features in Pieces.

### From Anywhere to Pieces Copilot

To navigate to Pieces Copilot:
1. Open the Power Menu
2. Search for "Go to Pieces Copilot" or type "copilot"
3. Select the command

You'll be taken directly to Pieces Copilot, ready to start a conversation or continue an existing chat.

![](https://storage.googleapis.com/pieces_static_resources/pro_tips/power_menu_copilot.png)

### From Anywhere to Workstream Activity

To navigate to Workstream Activity:
1. Open the Power Menu
2. Search for "Go to Workstream Activity" or type "workstream"
3. Select the command

You'll instantly be taken to your Workstream Activity timeline, where you can browse summaries, search, and interact with your work history.

![](https://storage.googleapis.com/pieces_static_resources/pro_tips/power_menu_workstream_activity.png)

### Quick Navigation Between Views

The Power Menu is context-aware—it shows suggested actions based on where you are in the app. When you're in Workstream Activity, you'll see Copilot-related commands prominently displayed, and vice versa.

**Pro Tip:** The Power Menu learns from your usage patterns. Frequently used navigation commands will appear higher in the list, making your most common jumps even faster.

## Pro Tips for Power Menu Navigation

### 1. Use Search, Don't Scroll
The Power Menu has a search function—use it! Type a few letters of what you're looking for (like "workstream" or "copilot") and the relevant commands will appear instantly.

### 2. Learn the Keyboard Shortcut
Set up and memorize the Power Menu keyboard shortcut. It's the fastest way to navigate—you can jump between views without touching your mouse.

### 3. Use It from Anywhere
The Power Menu works from any view in Pieces. Whether you're in Workstream Activity, Pieces Copilot, Pieces Drive, or Settings, you can always open the Power Menu to navigate anywhere else.

### 4. Context-Aware Commands
Remember that the Power Menu is dynamic. Commands relevant to your current view will appear first, making common workflows faster.

## Common Navigation Workflows

### Workstream Activity → Copilot Chat
1. Browse summaries in Workstream Activity
2. See something interesting
3. Open Power Menu
4. Select "Pieces Copilot"
5. Ask about the topic from the summary

### Copilot → Workstream Activity
1. Working in Pieces Copilot
2. Want to check what you did earlier
3. Open Power Menu
4. Select "Workstream Activity"
5. Browse or search summaries

### Quick Context Switch
1. In any view
2. Open Power Menu
3. Type "workstream" or "copilot"
4. Select the destination
5. Continue your work
